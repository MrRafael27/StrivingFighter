# Striving Fighter

Striving Fighter is a small mobile game currently in alpha version. It's my first game as a young developer, created in C# with Unity.

You can now try out the game with the first Snapshots by going to the "Releases" tab. Download the Android Package Kit file (*.apk) and open it on your Android device. Then follow the simple installation procedure.

Enjoy the game.

— Rafaël.
